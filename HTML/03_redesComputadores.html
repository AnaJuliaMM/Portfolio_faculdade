Tipos de comunicação

Camadas

  • Unicast: um dispositivo conversa unicamente com outro dispositivo
  • Multicast: um dispositivo se comunica com um grupo de dispositivos
  • Broadcast: um dispositivo se comunica com todos os participantes da rede

1

Meios de Transmissão

Meios

  • Simplex : um nó se comporta como transmissor e o outro como receptor de informação
  • Half Duplex: um nó pode se comportar tanto como receptor quanto transmissor, porém pode exercer apenas um papel por vez.
  • Full Duplex: um nó pode atuar como receptor e transmissor de dados simultâneamente

Protocolo TC/IP

O que é o Protocolo TC/IP?

O protocolo TC/IP rege o processo de transporte e comunicação numa rede de computadores.

2

Protocolos

Transmissão

  • HTTP (Hyper Text Transfer Protocol): protocolo para comunicação web, por exemplo, para transporte dos arquivos de uma página web
  • HTTPS (Hyper Text Transfer Protocol Security) protocolo que, semelhante ao HTTP, também atua na comunicação web, porém contém mais uma camada para adicionar segurança ao transporte
  • FTP (File Transfer Protocol) protocolo que atua no transporte de arquivos
  • FTPS (File Transfer Protocol Security): protocolo que atua no transporte de arquivos porém com uma camada extra de segurança

Transporte

  • UDP (User Datagram Protocol): protocolo que atua na camada de transporte e define que não é necessário a confirmação se o destino recebeu os arquivos com sucesso. São muito aplicados em transmissões de streaming.
  • TCP (Transmission Control Protocol): protocolo que garante a entrega dos pacotes de dados através de um sistema de confirmação entre emissor e o receptor.
