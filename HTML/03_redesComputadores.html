Modelo OSI

O que é o modelo OSI?

O modelo OSI é uma explicação dos processos que os dados passam para serem transportados de um computador à outro computador numa rede

Camadas

  • 7. Física: envia os dados em formato de bits
  • 6. Enlace: identifica os pacotes de dados com o MAC address
  • 5. Rede: compacta e identidica os pacotes de dados com endereço IP
  • 4. Transporte: define a forma de transporte dos pacotes (TCP e UDP)
  • 3. Sessão: inicia e encerra a comunicação entre os computadores da rede
  • 2. Apresentação: formata os dados a serem enviados e decodifica os recebidos
  • 1. Aplicação: Permite a interação do usuário com os dado recebidos. Ela trabalha com protocolos específicos como HTTP (web), SMTP (e-mails) e FTP (arquivos).

Protocolo TC/IP

O que é o Protocolo TC/IP?

O protocolo TC/IP rege o processo de transporte e comunicação numa rede de computadores.
