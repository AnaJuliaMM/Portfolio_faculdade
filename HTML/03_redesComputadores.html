Redes de Computadores

Redes de computadores é um conjunto de equipamentos conectados entre si que podem se comunicar trocando informações. Existem os seguintes tipos de redes:

  • LAN (Local Area Network): rede de alcance local
  • VLAN (Virtual Local Area Network): rede virtualmente segmentada em grupos menores
  • MAN (Metropolitan Area Network): rede de alcance metropolitano, onde se encaixam as SP (provedoras de sinais).
  • WAN (Wide Area Network): rede de alcance global

Dispositivos

Dispositivos Finais

São equipamentos que o usuário final utilizam, que posseum as placas de redes de origem ou destino dos pacotes de informação. São eles:

  • PC (personal computer) e Smartphones
  • Impressoras
  • Servidores. Exemplos:
    • Servidor Web: armazena os arquivos de uma página web
    • Servidor DNS: traduz o domínio de um site para seu IP correspondente
    • Servidor DNCH: Distribui IP automaticamente para os hosts
  • entre outros

Dispositivos Intermediários

São equipamentos utilizados na comunicação entre os nós finais e a fonte de informação, entre o receptor e o emissor de informação.

  • Switch e Hubs: responsáveis por ampliar as conexões de uma rede
  • Roteador: conecta as redes e calcula a rota do pacote de informações
  • Gateway: interface entre redes com arquiteturas diferentes

Conexões

As conexões podem ocorrer de forma física ou wireless. Como exemplo de conexão física, cita-se o cabo de par trançado. Duas são as formas de conexões físicas entre esses dispositivos.

  • Direta: entre dispositos de níveis diferentes
  • Cross-Over:entre dispositos de mesmos níveis

Modelo OSI

O que é o Modelo OSI?

É um modelo que explica os processos que os pacotes de dados passam na comunicação entre computadores de uma rede.

Camadas

  • 1. Física: onde os pacotes de dados são transportados na forma de bits
  • 2. Enlace: onde os pacotes são identificados com o endereço MAC
  • 3. Rede:onde os pacotes são identificados com o endereço IP
  • 4. Transporte:onde é definido o tipo de transporte de dados (UDP ou TCP)
  • 5. Sessão:onde é iniciada ou interrompida a comunicação entre os nós
  • 6. Apresentação: ondes os pacotes são formados para entrega
  • 7. Aplicação: onde o usuário tem acesso aos dados recebidos. Além disso, é onde atuam os protolos Http(páginas web), SMTP (e-mails) e FTP (arquivos)

Transmissão

Sobre a transmissão dos pacotes, estudamos os seguintes tópicos

  • Transmissão síncrona e assíncrona

Tipos de Casts

  • Unicast: um dispositivo conversa unicamente com outro dispositivo
  • Multicast: um dispositivo se comunica com um grupo de dispositivos
  • Broadcast: um dispositivo se comunica com todos os participantes da rede

IP e Máscara de Rede

conceito IP Máscara de Rede publico e privado

Protocolo de Internet(IP)

O IP é um identificador que é dado aos disposivos conectados a uma rede de computadores. Diferente dos endereços de MAC, que vem do fabricante, o IP de um dispositivo varia conforme a rede que ele se conecta. Sobre o assunto, estudamos:

  • O atual Ipv4 de 32 bits e o futuro Ipv6 de 64 bits
  • Existem faixas de IP reservadas pela IANA para diferentes redes, como as redes locais com o IP 10.0.0.0/8
  • IP reservados:
    • Primeiro(ex: 10.0.0.0/24): IP da rede
    • Último(ex: 10.0.0.255/24): IP de broadcast

Máscara de Rede

É um parâmetro da rede que divide os bytes do ip em identificadores de hots (dispositivos finais) e Redes. A placa de rede interpreta que onde existir um byte completo de número 1, ou o número decimal 255, é reservado para rede. E onde existe 0 é para hosts

  • Classe A:255.0.0.0 ou /8
  • Classe B:255.255.0.0 ou /16
  • Classe C:255.255.255.0 ou /24
  • Existem máscaras que fogem dos números exatos, por exemplo: 255.255.255.224 ou /27

IP privado e público

  • IP privado:ip que um dispositivo possui dentro de uma rede
  • IP público:IP da interface do roteador que estabelece conexão externa e representa toda a rede
  • Protocolo NAT:é o processo que o roteador realiza de traduzir o IP publico para o privado e vice-versa
  • Servidor DHCP:é o dispositivo que distriu automaticamente op IP´s de uma rede.
