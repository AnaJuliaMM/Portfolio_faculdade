Javascript

Introdução Teórica

Começamos estudando a história do desenvolvimento da linguagem Javascript para responder três perguntas apresentadas pelo professor: primeiro, “javascript tem alguma relação com Java?”; segundo, “qual a diferença entre javascript e ECMAScript?” ; e terceiro, “o que é e para que serve o node.js?” . A partir da aula, pudemos concluir que não há uma relação entre Java e Javascript, que ECMAScript é a versão padronizada da linguagem e que Node.js é o motor que permiti a execução do JS.
Dando continuidade, para avançarmos nas aulas práticas, realizamos a instalação do node.js e executamos alguns comandos para demonstração da utilização do console no VSCODE e em um navageador web. Dessa forma, o ambiente de aprendizado foi preparado para o estudo dos conceitos de programação e desenvolvimento de projetos.

Variáveis

Variáveis são espaços na memória reservados para armazenar valores gerados durante a execução de um programa. Sobre o assunto, estudamos os seguintes tópicos:

  • Sintaxe de declaração e inicialização
  • Regras de nomeclatura de variáveis
  • Tipos de Variáveis(Number, String e Boolean)
  • Concatenação de String

    • Operadores

    Operadores são símbolos que permitem realizar operações com variáveis. Em JS existem os seguintes tipos:

  • Atribuição (=)
  • Aritmética (+, +=, -, *, /, **, %, ...)
  • Condicional (==, >, !=, ===, ...)
  • Lógicos (&&, ||, !)
  • Ternário

    • Modelo de Objeto de Documento

    O D.O.M é a estrutura hierarquica dos elementos de uma página, e via comandos do javascript pode-se acessar e manipular esses elementos num programa. Tópicos Abordados:
    • Árvore do D.O.M
    • Métodos de acesso
    • Eventos

    Desvios Condicionais

    Estas estruturas possibilitam alterar a ordem padrão de execução dos código ao atrelar uma condição à execução de um bloco de código, realizando assim um desvio.

    • Simples (if)
    • Composto (if... else)
    • Encadeado (if... else if... else)
    • switch case

    Laços de Repetição

    Comandos que permitem a repetição da execução de um bloco de código. São eles:

    • while
    • do while
    • for

    1 2 3 4

    Desafio 1: Restrição de Idade

    O objetivo do programa é receber a entrada da idade do usuário. Enquanto a idade inserida for menor que 18, um alerta será emitido ao usuário e ele poderá inserir novas entradas. Quando a idade for maior ou igual à 18, um aviso de permissão de acesso será lançado. As seguintes estruturas foram utilizadas:

    • do while
    • If...Else
    • Comando DOM: window.alert("Mensagem")

    Desafio 2: Identificador de números ímpares e pares

    O objetivo do programa é receber a entrada de um número inteiro e notificar ao usuário se é um número ímpar ou número par. As seguintes estrutura foram utilizadas:

    • If...Else
    • Módulo de Divisão(%)
    • Comando DOM: window.alert("Mensagem")

    Desafio 3: Introdução ao D.O.M

    O objetivo do programa é mudar a cor e a legenda do botão assim que o cursor do mouse passar em cima desse elemento. Dessa forma, foram utilizados os seguintos comandos de DOM:

    • Método de acesso: document.getElementById("id")
    • Eventos: elemento.addEventListener ("mouseover", nomeFuncao)
    • Funções(ex: mudarCor())
    • Estilização: elemento.style.color = "red" e elemento.innerText = "clicou"

    Desafio 4: Interagindo com data

    Uma mensagem é gerada no console contendo o dia da semana e uma mensagem de "bom dia", "boa tarde" ou "boa noite" de acordo com o horário do dia. Foi trabalhado os seguintes conteúdos:

    • If...else
    • Switch case
    • Comandos: new Date(), .getHours(), .getDate()
