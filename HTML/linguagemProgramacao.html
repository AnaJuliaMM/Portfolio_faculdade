HTML 5: Html Semântico

O HTML5 é a mais recente versão do html, cujo pilar é a semântica . No lugar da utilização de inúmeras tags div com classes e identificadores, há opção de utilizar tags específicas que correspondem à função daquele trecho de código

Exemplos
  • header: cabeçalho
  • main: conteúdo principal
  • section:seção dentro de um corpo de texto
  • entre outros
