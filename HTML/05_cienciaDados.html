Estatística

A princípio aprendemos sobre o conceito de população e amostragem, como ilustra a figura. E então, estudamos os tipos de amostra:

  • Aleatória: cada elemento da população tem a mesma possibilidade de serem escolhidos.
  • Sistemática: é feito uma seleção aleatória do primeiro elemento da amostra e em seguidas os demais são escolhidos através de intervalos fixos.
  • Estratificada: separa os elementos em grupos homogêneos e em seguida é feita uma amostra aleatória em cada grupo (estrato).

Escalas de Medição

Um dos pilares da análise de dados é a pesquisa, e para extrair conclusões dessas consultas existem métodos para medir as respostas e, assim, extrair relatórios. São eles:

  • Escala Nominal: as variáveis são opções de nomeação (ex: cores)
  • Escala Ordinal: as variáveis são valores categorizados(ex: nível de satisfação)
  • Escala Intervalar: nota em um intervalo de valores (ex: 1 a 10)
  • Escala Razão: as variáveis são intervalos (ex: faixa etárias: 1-10 anos, 20-30 anos, ...)

Tendências Centrais

Moda Média Aritmética Mediana Percentil Phyton

Medidas de tendências centrais são um recurso matemático para representar um conjunto de valores por um único valor.

Moda

O(s) termo(s) que aparece(m) com mais frequência num conjunto. Por exemplo:

A = [1, 4, 5, 7, 4, 9, 10, 4, 4] | Moda = [4]

Um conjunto pode ser:

  • Modal: possui 1 moda
  • Bimodal: possui 2 moda
  • Amodal: não possui moda

Média Aritmética

É o valor médio de um conjunto.

Mediana

É o termo central de um conjunto ordenado. Por exemplo:

A = [1, 2, 3, 4, 5]


B = [1, 2,3, 4] - Mediana= 2+3/2 = 2,5

Percentil

Indica a posição em porcentagem de um valor dentro de seu conjunto ordenado. Ainda, existe o quartil que divide o conjunto nos seguintes intervalos:

  • Primeiro Quartil - 25%
  • Segundo Quartil - 50%
  • Terceiro Quartil - 75%

Métodos em Phyton

  • moda: dataFrame.mode()
  • mediana: dataFrame.median()
  • media: dataFrame.mean()
  • percentil/quartil: dataFrame.quantile()

5

Medidas de Dispersão

Amplitude Variância Desvio Padrão Phyton

São métodos matemáticos que evidenciam a variação dos valores de um conjunto.

Amplitude

É a diferença entre o menor e o maior valor

Variância

Indica o quanto os valores de um conjunto estão dispersos de sua média.

Desvio Padrão

Fornece uma valor para determinar o intervalo com maior concentração de dados

Métodos em Phyton

  • variâncis: dataFrame.var()
  • desvio padrão: dataFrame.std()

7

Teste de Hipótese

São métodos para atestar o quanto o resultado de uma amostra pode representar uma população. Estudamos os seguintes métodos

  • Qui Quadrado: teste de dependência das variáveis
  • Kolmogorov-Smirnov: verificar se duas amostra vem da mesma distribuição
  • Shapiro-Wilk: verificar se a amostra vem de uma distribuição normal
  • Anderson Darling: mede o quão bem os dados se adequam a uma destribuição específica
